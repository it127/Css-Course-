CSS Course

Learn to style and layout your web pages with CSS.

Course Overview

This CSS course is designed to help you master the art of styling and layout in web development. You'll learn the essentials of CSS, including selectors, properties, and layout techniques. By the end of this course, you'll be able to create visually appealing and responsive web pages.

CSS Basics

1. Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to style and layout HTML documents. CSS can control the layout, colors, fonts, and other visual aspects of a web page.

2. CSS Syntax

CSS syntax consists of a selector and a declaration block. The selector points to the HTML element you want to style, and the declaration block contains one or more declarations separated by semicolons.

selector {
    property: value;
}

3. CSS Selectors

CSS selectors are used to select the HTML elements you want to style. Common selectors include element selectors, class selectors, and ID selectors.

/* Element Selector */
p {
    color: red;
}

/* Class Selector */
.red-text {
    color: red;
}

/* ID Selector */
#main-header {
    color: blue;
}

4. CSS Properties

CSS properties are used to set the styles for HTML elements. Some common properties include color, background-color, font-size, and margin.

p {
    color: red;
    background-color: yellow;
    font-size: 16px;
    margin: 10px;
}

Advanced CSS Topics

1. CSS Layout

CSS provides several layout techniques, including flexbox and grid, to create responsive and flexible layouts.

/* Flexbox Example */
.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

/* Grid Example */
.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

2. CSS Animations and Transitions

CSS animations and transitions can be used to add dynamic effects to your web pages. Transitions are used for simple effects, while animations are used for more complex effects.

/* Transition Example */
button {
    background-color: #4a6bff;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    transition: background-color 0.3s;
}

button:hover {
    background-color: #6c5ce7;
}

/* Animation Example */
@keyframes slide-in {
    from { transform: translateX(-100%); }
    to { transform: translateX(0); }
}

.content {
    animation: slide-in 1s;
}

3. CSS Variables (Custom Properties)

CSS variables (custom properties) allow you to define reusable values that can be used throughout your stylesheet.

:root {
    --primary-color: #4a6bff;
    --secondary-color: #6c5ce7;
}

button {
    background-color: var(--primary-color);
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
}

button:hover {
    background-color: var(--secondary-color);
}

4. CSS Grid

CSS Grid is a two-dimensional layout system that allows you to create complex layouts with rows and columns.

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(2, 1fr);
    gap: 10px;
}

.grid-item {
    background-color: #4a6bff;
    color: white;
    padding: 20px;
    text-align: center;
}

5. CSS Flexbox

CSS Flexbox is a one-dimensional layout system that makes it easy to align and distribute space among items in a container.

.flex-container {
    display: flex;
    justify-content: center;
    align-items: center;
    flex-wrap: wrap;
    gap: 10px;
}

.flex-item {
    background-color: #4a6bff;
    color: white;
    padding: 20px;
    text-align: center;
    flex: 1 1 150px;
}

6. CSS Pseudo-classes and Pseudo-elements

CSS pseudo-classes and pseudo-elements are used to style elements based on their state or position in the document.

/* Pseudo-class Example */
a:hover {
    color: #4a6bff;
}

/* Pseudo-element Example */
p::first-letter {
    font-size: 24px;
    font-weight: bold;
}

7. CSS Media Queries

CSS media queries are used to apply different styles for different media types or device characteristics. They are essential for creating responsive designs.

@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}

8. CSS Units

CSS units are used to specify the size of elements. Common units include pixels (px), em, rem, and percentages (%).

.element {
    width: 200px; /* Fixed width */
    font-size: 16px; /* Fixed font size */
    padding: 1em; /* Relative to the font size of the element */
    margin: 5%; /* Relative to the width of the parent element */
}

9. CSS Box Model

The CSS box model is a box that wraps around every HTML element. It consists of margins, borders, padding, and the actual content. Understanding the box model is crucial for layout and spacing.

.box {
    width: 200px;
    height: 100px;
    padding: 10px;
    border: 5px solid black;
    margin: 15px;
}

10. CSS Specificity

CSS specificity determines which CSS rule is applied by the browser when multiple rules apply to the same element. Specificity is calculated based on the number of ID, class, and element selectors.

#id .class element {
    color: red;
}

11. CSS Inheritance

CSS inheritance is the process by which certain CSS properties are passed down from parent elements to their child elements. Properties like color and font-size are inherited by default.

.parent {
    color: blue;
    font-size: 16px;
}

.child {
    /* Inherits color and font-size from .parent */
}

12. CSS Preprocessors

CSS preprocessors like SASS and LESS allow you to use features that are not available in standard CSS, such as variables, nesting, and mixins. They compile into standard CSS.

$primary-color: #4a6bff;

.container {
    background-color: $primary-color;
    padding: 20px;

    .item {
        color: white;
        padding: 10px;
    }
}

13. CSS Grid and Flexbox for Responsive Design

Combining CSS Grid and Flexbox can help you create highly responsive and flexible layouts. Grid is great for two-dimensional layouts, while Flexbox is ideal for one-dimensional layouts.

.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    gap: 10px;
}

.grid-item {
    background-color: #4a6bff;
    color: white;
    padding: 20px;
    text-align: center;
}

@media (max-width: 768px) {
    .flex-container {
        flex-direction: column;
    }
}

Practical Examples

1. Styling a Simple Web Page

Here's a simple web page styled with CSS:

<!DOCTYPE html>
<html>
<head>
    <title>My Styled Web Page</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f4f4f9;
            margin: 0;
            padding: 0;
        }
        .container {
            max-width: 800px;
            margin: 0 auto;
            padding: 20px;
        }
        h1 {
            color: #333;
        }
        p {
            color: #666;
        }
        a {
            color: #4a6bff;
            text-decoration: none;
        }
        a:hover {
            color: #6c5ce7;
        }
    </style>
</head>
<body>
    <div class="container">
        <h1>Welcome to My Styled Web Page</h1>
        <p>This is a paragraph of text.</p>
        <a href="https://example.com">Visit Example.com</a>
    </div>
</body>
</html>

2. Creating a Responsive Layout with Flexbox

Here's an example of a responsive layout using CSS Flexbox:

<style>
    .flex-container {
        display: flex;
        justify-content: center;
        align-items: center;
        flex-wrap: wrap;
        gap: 10px;
    }

    .flex-item {
        background-color: #4a6bff;
        color: white;
        padding: 20px;
        text-align: center;
        flex: 1 1 150px;
    }

    @media (max-width: 768px) {
        .flex-container {
            flex-direction: column;
        }
    }
</style>
<div class="flex-container">
    <div class="flex-item">Item 1</div>
    <div class="flex-item">Item 2</div>
    <div class="flex-item">Item 3</div>
</div>

3. Creating a Grid Layout

Here's an example of a grid layout using CSS Grid:

<style>
    .grid-container {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
        grid-template-rows: repeat(2, 1fr);
        gap: 10px;
    }

    .grid-item {
        background-color: #4a6bff;
        color: white;
        padding: 20px;
        text-align: center;
    }
</style>
<div class="grid-container">
    <div class="grid-item">Item 1</div>
    <div class="grid-item">Item 2</div>
    <div class="grid-item">Item 3</div>
    <div class="grid-item">Item 4</div>
    <div class="grid-item">Item 5</div>
    <div class="grid-item">Item 6</div>
</div>
